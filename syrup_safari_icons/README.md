# Syrup Safari Icon Set

## Overview
A complete artisanal, botanical-style icon set for the Syrup Safari mobile app.

## Design Philosophy
- **Style**: Organic, artisanal, handcrafted aesthetic
- **Technique**: Hybrid linework + watercolor fills with soft imperfect edges
- **Colors**: Earthy botanical palette (sage, eucalyptus, moss, soft peach, lavender, citrus)
